Yoda

By configuring a simple YAML configuration file, you can easily manipulate docker containers needed for your projects. Instead of writing bash scripts over and over. Let Yoda do the tedious work for you. A few features include:

  • Docker cli wrapper, meaning you can use bash scripting just about everywhere!
  • Build, pull, remove, containers and all of their dependancies.
  • Many ways to get up and running quickly, by summoning configuration files that others have shared!
  • Have a great configuration file? Share it with others!
  • Recursively seek out .yoda configuraiton files, and try to start containers. Excellent for dev enviornments where you have multiple projects!
  • Need to perform different actions based on differnet enviornments? You can do it all within the same configuration file.

And a few more goodies ...
